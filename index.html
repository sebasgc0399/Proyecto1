Apuntes de Git Bash

COMANDOS GIT BASH:

git init (Configurar Git en el proyecto)

git add . (Agregar todos los archivos)
git reset . (Retroceder los git add)

git commit -m "Comentario" (Checkpoints)
git checkout -- . (Restablecer Local lo del Repositorio)
git log (Listado de Commits)
git commit --amed (Editar Comentario de mi ultimo Commit)

git branch (Consultar Ramas Existentes)
git checkout Nombre (Cambiar Rama)
git checkout -b rama-Nombre (Crear Rama)
git branch -d rama-Nombre (Eliminar Rama)

git push (Subir al Repositorio)
git pull (Actualizar Local de acuerdo al Repositorio)
